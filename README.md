# Site de Viagens

Recriação de um site de viagens utilizando HTML e CSS.

---

### Descrição

Projeto simples com o objetivo de praticar a criação de páginas web, incluindo uma página principal e uma página de destinos com imagens.

---

### Tecnologias

* HTML
* CSS

---

### Páginas

* index.html – Página inicial
* destinos.html – Página de destinos

---

### Estrutura

```
/projeto
  ├── index.html
  ├── destinos.html
  ├── style.css
  ├── imagens
 
```

---

### Funcionalidades

* Navegação entre páginas
* Exibição de destinos com imagens
* Layout organizado
* Formulário de contato
